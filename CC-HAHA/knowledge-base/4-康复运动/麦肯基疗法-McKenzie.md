# 麦肯基方法 (McKenzie Method / MDT)

来源: Wikipedia (CC BY-SA)

## 概述
麦肯基方法由新西兰物理治疗师 Robin McKenzie 于20世纪50年代末开发，1981年正式推出"机械诊断与治疗（MDT）"概念体系。MDT不按解剖学分类，而是根据**患者的临床表现**进行亚组分类。

## 核心原理：向心化（Centralisation）
向心化是指疼痛症状从偏离脊柱中线的位置向中线迁移。症状向腰部中心迁移视为**进展的标志**。

- **有益的运动**：能够产生向心化的动作
- **有害的运动**：使疼痛远离脊柱中线的动作（外周化）

## 动作类型
- 脊柱后伸（spinal extension）
- 脊柱屈曲（flexion）
- 侧方移位（lateral shift）
- 后伸联合侧方移位
- 屈曲伴旋转运动

**患者对动作的反应决定所规定的运动方案。** 患者通常对脊柱后伸反应良好，但该方法并不回避其他方向。

## 有效性证据

| 疼痛类型 | 证据 |
|----------|------|
| 慢性下背痛 | 中等至高质量证据表明MDT在减轻疼痛和残疾方面优于某些其他康复干预，但效果大小为小到中等 |
| 急性下背痛 | MDT**并不优于**其他康复干预措施 |
| 亚急性下背痛 | 未显示能显著减轻症状或残疾 |

- 2019年系统评价：MDT可短期减轻慢性下背痛、长期改善功能，但大多数研究存在方法学缺陷
- 2022年荟萃分析：基于分类的方法（MDT为其一）可能略优于其他方案，但证据不足以支持优先使用
- **中线强化练习并不比传统屈伸练习更有帮助**

## 关键提示
- 应促进椎间盘向心化的练习
- 避免促进椎间盘外周化的练习
- 任何导致疼痛**远离脊柱中线**的动作应被避免
- 1956年McKenzie发现一位患者向后弯曲五分钟后症状改善，此法由此而生
