# 日常姿势与工效学

> 来源：Wikipedia (CC BY-SA)，综合整理

## 久坐的影响

- 每天每多坐1小时，全因死亡率增加约3%
- 65岁以上人群久坐可达清醒时间的80%
- 久坐与肥胖、心血管病、糖尿病、部分癌症相关

### 工位调整
- 屏幕顶部与眼睛平齐，避免低头
- 手肘约90°，手腕平直
- 腰部有支撑，脚平踩地面
- 可尝试坐站交替办公桌

## 搬重物补充要点

- **推比拉对背部更友好**（推用腿发力，拉容易弓背）
- 搬东西转身时**肩、髋、脚同步转动**，保持物品在前方，不要只扭腰
- 具体脊柱生物力学和搬物技巧详见《01-脊柱生物力学.md》

## 常见姿势偏差

| 偏差 | 表现 |
|------|------|
| 头前伸 | 耳垂在肩峰垂直线前方 |
| 圆肩 | 肩部向前旋转 |
| 驼背 | 胸椎后凸过度 |
| 平背 | 胸椎曲度减小 |
| 腰椎前凸过度 | 骨盆前倾、肚子前挺 |

## 自我检查：靠墙站立

后脑勺、肩胛骨、臀部、脚后跟贴墙：
- 颈后与墙间距约**2指**
- 腰后与墙间距约**一掌厚**
- 差距过大或过小提示相关节段曲度异常
