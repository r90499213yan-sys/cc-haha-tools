# 人体力学与脊柱负荷

来源: Wikipedia (CC BY-SA)，综合多篇文献

## 中立位脊柱

从侧面看，健康脊柱呈 S 形曲线：
- **颈椎**：向前弯曲（前凸）
- **胸椎**：向后弯曲（后凸）
- **腰椎**：向前弯曲（前凸）
- **骶骨/尾骨**：位于骨盆之间

中立位不是"完全竖直"，骨盆本身就有轻微前倾。

## 不同姿势下的椎间盘压力（腰椎 L3 节段）

> 以下数据来自 Nachemson 等人经典研究，是被引用最多的脊柱负荷参考：

| 姿势 | 椎间盘压力（相对值） |
|------|---------------------|
| 仰卧 | **25%**（最低，基线） |
| 侧卧 | 75% |
| 直立站立 | 100% |
| 坐直（无靠背） | 140% |
| 站立前倾 | 150% |
| 坐姿前倾 | **185%** |
| 坐姿前倾 + 提重物 | **275%**（最高） |

**关键结论**：
- 坐姿比站姿对椎间盘压力**大约高 40%**
- 前倾坐姿是最糟糕的日常姿势
- 躺下是给椎间盘"减压"的最佳方式

## 正确的搬重物技巧

### 基本原则
1. 先评估重量，太重找人或用工具
2. 双脚站稳，地面不滑
3. **屈膝下蹲，保持背部挺直**
4. 双手抓稳物品
5. **用腿部发力站起**，不是用腰
6. 物品贴近身体（胸口到大腿中部之间）
7. 放下时同样屈膝，保持背直

### ⚠️ 常见错误
- 直腿弯腰搬东西 → 椎间盘压力暴增
- 搬起后扭转躯干 → 旋转+负荷是椎间盘突出高危动作
- 手臂伸直远离身体 → 力臂加长，腰椎负荷成倍增加

### 推 vs 拉
**推比拉对背部更友好。** 推的时候用腿发力，拉的时候容易弓背。

### 转身
搬东西转身时，**肩、髋、脚同步转动**，保持物品始终在前方。不要只扭腰。

## 久坐的人体力学问题

- 每天每多坐 1 小时，全因死亡率增加约 3%
- 65 岁以上人群久坐时间可达清醒时间的 **80%**
- 久坐与肥胖、心血管病、糖尿病、某些癌症相关

### 工位调整建议
- 屏幕顶部与眼睛平齐，避免低头
- 手肘约 90°，手腕平直
- 腰部有支撑，脚平踩地面
- 可使用**坐站交替办公桌**（部分研究显示有益，但也有大量研究未发现明显健康改善）

## 姿势与背痛的关系

⚠️ **一个重要但常被忽略的事实**：

目前医学研究**尚未证实不良姿势与下背痛之间存在确定的因果关系**（Swain et al., 2020 系统综述）。关于"最佳坐姿"在物理治疗师之间也没有统一意见（O'Sullivan et al., 2012）。

这意味着：
- 不能说"你腰痛就是因为你坐姿不好"
- 但保持中立位仍然有利于平衡、力量和柔韧性
- 姿势建议应作为综合康复的一部分，而非唯一答案

## 常见姿势偏差

| 偏差 | 表现 |
|------|------|
| 头前伸 | 耳垂在肩峰垂直线前方 |
| 圆肩 | 肩部向前旋转 |
| 驼背 | 胸椎后凸过度 |
| 平背 | 胸椎曲度减小 |
| 腰椎前凸过度 | 骨盆前倾、肚子前挺 |
| 脊柱侧弯 | 前后方向上的侧向弯曲 |

## 如何自我检查

靠墙站立，后脑勺、肩胛骨、臀部、脚后跟贴墙：
- 颈后与墙间距约 2 指
- 腰后与墙间距约一掌厚
- 差距过大或过小提示相关节段曲度异常
