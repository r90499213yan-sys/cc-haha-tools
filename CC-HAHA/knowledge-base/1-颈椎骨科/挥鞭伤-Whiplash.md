# 挥鞭伤 (Whiplash)

来源: Wikidoc (CC BY-SA 3.0)

## 概述
挥鞭伤是颈部扭伤的通用术语，由颈椎过伸/过屈损伤引起。最常见于**机动车追尾事故**，也可由自行车/马匹跌落、过山车、滑雪事故等导致。

## 症状
- 颈部和背部疼痛及酸痛
- 肩部牵涉痛
- 手臂和腿部感觉障碍（针刺感）
- 头痛
- **症状可直接出现在伤后，但通常数天后才感觉到**
- 最常累及颈部和背部中部

## QTF分级（Québec Task Force）

| 分级 | 表现 |
|------|------|
| **1级** | 仅主诉颈痛、僵硬或压痛——医生检查无体征 |
| **2级** | 颈部主诉 + 医生发现活动范围减小和点压痛 |
| **3级** | 活动范围减小 + 神经体征（深腱反射减弱、无力、失眠、感觉缺失） |
| **4级** | 颈部主诉 + 骨折、脱位或脊髓损伤 |

## 诊断
- X线**看不到软组织损伤**（如椎间盘撕裂），需用MRI
- 严重损伤（如翼状韧带撕裂）需要**功能成像**才能显示

## 后果
- 大多数人仅经历数天轻微疼痛
- 严重者可致头部或颈椎活动受限的严重残疾，有时伴持续性疼痛
- 可**加重先前存在的疾病**如颈椎病等退行性改变
