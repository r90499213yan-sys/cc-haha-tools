# 皮节 (Dermatomes) — 神经定位关键参考

来源: Wikipedia (CC BY-SA)

## 定义
皮节是由特定脊神经背根的传入神经纤维主要支配的一片皮肤区域。共有8对颈神经（C1例外，无皮节）、12对胸神经、5对腰神经和5对骶神经。

**精确的神经支配区域因人而异，如同指纹一般独特。**

## 临床意义
- 沿皮节分布的症状（疼痛、皮疹）提示相关神经根病变
- **牵涉痛**：内脏感觉纤维受刺激时，疼痛被感知为来自同一脊髓节段的对应皮节（如心肌缺血→左臂痛）
- **带状疱疹**：潜伏在神经节内的病毒沿皮节模式产生疼痛/皮疹

## 颈腰椎常见神经根定位速查

**颈椎：**
- C5 → 肩外侧 + 三角肌区
- C6 → 前臂桡侧 + 拇指
- C7 → 中指（最常见颈椎间盘突出节段）
- C8 → 前臂尺侧 + 小指

**腰椎：**
- L4 → 小腿内侧 → 内踝
- L5 → 小腿外侧 → 足背 → 大脚趾
- S1 → 小腿后侧 → 足外侧 → 小趾 → 足底
