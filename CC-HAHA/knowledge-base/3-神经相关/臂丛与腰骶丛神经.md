# 臂丛与腰骶丛神经

来源: Wikipedia (CC BY-SA)

## 臂丛神经 (Brachial Plexus)

### 结构：5根→3干→6股→3束→5终末支
"Rich Tourists Drink Cold Beer" = Root→Trunk→Division→Cord→Branch

### 根 (C5-T1)
- C5+C6 → 上干
- C7 → 中干
- C8+T1 → 下干

### 关键终末分支

| 神经 | 节段 | 运动 | 感觉 |
|------|------|------|------|
| 肌皮神经 | C5-C7 | 肱二头肌、肱肌 | 前臂外侧 |
| 腋神经 | C5-C6 | 三角肌 | 肩外侧 |
| 桡神经 | C5-T1 | 肱三头肌、前臂伸肌 | 前臂后侧 |
| 正中神经 | C5-T1 | 前臂屈肌、鱼际肌 | 手外侧 |
| 尺神经 | C7-T1 | 手内在肌 | 手内侧 |

### 斜方肌和腋窝皮肤分别由副神经和肋间臂神经支配（例外）

### 损伤类型
- **上干损伤 (Erb-Duchenne)**：C5-C6，三角肌+肱二头肌瘫痪，"服务员小费手"
- **下干损伤 (Klumpke)**：C8-T1，手内在肌瘫痪，"爪形手"
- **肩部向下推+头向上拉** → 臂丛牵拉伤（车祸+新生儿最常见）
- **运动损伤"stinger"**：美式橄榄球等，固定臂丛在肩垫与肩胛之间受压

## 腰骶神经丛 (Lumbosacral Plexus)

### 三部分
1. **腰丛** (L1-L4)：股神经、闭孔神经、股外侧皮神经
2. **骶丛** (L4-S4)：坐骨神经（全身最粗大神经）
3. **阴部丛** (S2-S4)

### 坐骨神经分支
- **腓总神经** (L4-S2)：足背屈、足趾伸展、足外翻
- **胫神经** (L4-S3)：足跖屈、足趾屈曲、足内翻
- **腓肠神经**：足外侧感觉

### 关键神经根
- L2-L4 → 股神经 → 伸膝
- L4-S2 → 腓总神经 → 足背屈（抬脚尖）
- L5-S2 → 胫神经 → 足跖屈（踮脚）
- L4-S3 → 坐骨神经 → 下肢后侧整体
