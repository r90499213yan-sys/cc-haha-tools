# 肌节 (Myotomes) — 运动功能定位

来源: Wikipedia (CC BY-SA)

## 定义
肌节是由单个脊神经支配的一组肌肉。每条脊神经根的运动纤维支配一组特定肌肉，肌节是皮节的运动对应物。

## 临床意义
肌节测试是神经系统检查的重要组成部分。通过**等长抗阻肌肉测试**可为医生提供关于脊髓病变可能水平的信息。检查者主要观察特定肌群是否存在肌无力。

结果可能提示：脊髓神经根损伤或**椎间盘突出压迫脊神经根**。

## 颈椎间盘突出常见肌节定位
- **C5-C6 突出** → C6受压 → 肘屈曲 + 腕伸展无力
- **C6-C7 突出** → C7受压 → 肘伸展 + 腕屈曲无力（最常见突出节段）
- **C7-T1 突出** → C8受压 → 拇指伸展 + 手指活动无力

## 腰椎间盘突出常见肌节定位
- **L3-L4 突出** → L4受压 → 踝背屈无力（足下垂）
- **L4-L5 突出** → L5受压 → 大脚趾背伸无力
- **L5-S1 突出** → S1受压 → 踝跖屈无力（不能踮脚）
