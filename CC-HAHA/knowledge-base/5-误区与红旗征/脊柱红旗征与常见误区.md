# 脊柱红旗征与常见误区

综合来源: Wikidoc, Wikipedia (CC BY-SA/BY-SA 3.0)

## ⚠️ 脊柱疼痛红旗征 (Red Flags)
出现以下任一症状，必须建议就医，**不可自行康复训练**：

### 绝对红旗征
1. **大小便失禁/潴留** → 马尾综合征可能
2. **鞍区麻木**（会阴/肛周感觉丧失） → 马尾综合征
3. **进行性下肢无力**（尤其是双侧） → 脊髓压迫
4. **不明原因体重下降** → 恶性肿瘤可能
5. **发热 + 背痛** → 感染（骨髓炎/硬膜外脓肿）
6. **严重创伤后突发背痛**（如车祸、高处坠落）→ 骨折
7. **夜间痛醒**（非活动相关，休息不缓解）→ 肿瘤/感染
8. **有癌症病史** → 转移可能
9. **长期使用皮质类固醇** → 骨质疏松压缩性骨折
10. **骨质疏松/多发性骨髓瘤患者背痛加重**

### 相对红旗征（需谨慎评估）
- 年龄 <20岁 或 >55岁（首次发作）
- 胸椎疼痛（胸椎椎间盘突出极少见）
- 静脉药物滥用史
- 免疫抑制状态

## 🔬 常见误区纠偏

### 误区1："腰突必须卧床休息"
**事实**：90%的椎间盘突出在无特殊干预下恢复。卧床休息应**控制在1-3天**，之后应逐步恢复正常活动（以耐受为度）。长期卧床反而**延迟恢复**。

### 误区2："小燕飞/山羊挺身适合所有腰突患者"
**事实**：后伸运动可能对部分患者有益（麦肯基向心化），但**对某些亚型会加重**。应评估患者对运动方向的反应——如果动作使疼痛向外周放射（外周化），应立即停止。

### 误区3："MRI/X光检查一下更放心"
**事实**：
- 高达85%的下背痛**无法确定生理性病因**
- 50%无症状志愿者颈椎MRI有椎间盘突出
- 过早影像学检查可能导致过度治疗，"可能对患者有害"

### 误区4："核心训练是康复特效药"
**事实**：核心稳定性训练效果**与一般运动相同**，无额外预防损伤或提升表现的效果。不应夸大其"特效"。

### 误区5："针灸/推拿能治好颈椎病"
**事实**：针灸对慢性背痛有**小型短期获益**，不优于其他治疗。推拿有小型获益，**未证实优于传统止痛抗炎药**。

### 误区6："加巴喷丁/普瑞巴林能治坐骨神经痛"
**事实**：加巴喷丁类药物对坐骨神经痛**无效**。

### 误区7："硬膜外注射能治好腰突"
**事实**：硬膜外类固醇注射**仅提供短期缓解**，无长期获益。FDA未确认其有效性和安全性。

### 误区8："椎间盘滑脱了能推回去"
**事实**："椎间盘滑脱"是误称。椎间盘与相邻椎体**牢固连接，不会滑出**。手法治疗不能"推回"椎间盘。

### 误区9："不弯膝盖直接弯腰没事"
**事实**：直接弯腰不弯膝对下背部造成**严重冲击**，拉伤腰骶区域和多裂肌。

### 误区10："旋转运动很安全"
**事实**：旋转运动下的压力对椎间盘危害较大——弯腰偏向一侧以不当角度捡东西**更容易导致椎间盘突出**。
