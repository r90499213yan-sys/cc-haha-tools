# 脊椎滑脱 (Spondylolisthesis)

来源: Wikidoc (CC BY-SA 3.0)

## 概述
Spondylolisthesis 描述椎体相对于下方椎体的移位。"spondylo"=椎骨，"olisthesis"=滑移。最常见于腰椎。

绞刑者骨折（Hangman's fracture）：C1椎体因C2椎弓根骨折而相对于C2向前移位。

## 峡部裂（Spondylolysis）
- 峡部裂是导致滑脱**最常见的原因**
- 椎弓峡部缺损约在**6岁开始出现**，16岁后发病率下降，成年后很少再发
- 缺损是**发育性而非先天性**——源于未愈合的小应力性骨折形成慢性骨不连
- **反复过伸运动的运动员中更常见**：体操、芭蕾、美式足球
- 有家族聚集性。因纽特人中滑脱发生率为30–50%

## Wiltse 分型（五型）

### 1. 发育不良型
真性先天性滑脱，腰骶结合部畸形、小关节发育不良。极罕见但进展迅速，常伴严重神经功能缺损。

### 2. 峡部型（最常见）
美国人群患病率约5%-7%。约90%为低度（滑移<50%），10%为高度（>50%）。

### 3. 退变型
老年人疾病，继发于小关节炎及重塑。65岁以上白人女性患病率30%，非洲裔美国女性60%。合并腰椎管狭窄时可加重神经源性跛行。**退变性滑脱伴椎管狭窄是老年人脊柱手术最常见适应证之一**。

### 4. 创伤型
极罕见，与小关节或椎弓峡部急性骨折相关。

### 5. 病理型
继发于转移瘤或代谢性骨病（Paget病、结核、巨细胞瘤等）。

## Meyerding 分级（侧位X线）
- 1级：0–25%
- 2级：25–50%
- 3级：50–75%
- 4级：75–100%
- >100%：脊椎完全脱位（spondyloptosis）

## 症状
- 背部僵硬、**腘绳肌紧张**，导致姿势和步态改变
- 前倾姿势或腰椎前凸加重
- 严重时步态呈"摇摆样"
- 常见臀肌萎缩
- 下背部广泛性疼痛，间歇性放射痛从臀部至大腿后侧/小腿
- 坐骨神经痛可延伸至膝以下、足部
- 坐位及站起困难，**咳嗽和打喷嚏可加剧疼痛**
- 活动量增加后因软组织炎症导致随后数日疼痛加重，**休息可缓解**

### 低度峡部型（<50%）
活动相关性背痛，常伴根性症状。生物力学研究发现46%患者滑脱水平出现反常运动（vs 0%对照）。

### 退变性滑脱
无神经弓缺损，椎体前移导致中心椎管狭窄（"餐巾环效应"）。症状类似腰椎管狭窄：神经源性跛行或根性痛。典型表现：双侧腿部后侧疼痛，活动加重，坐位或前屈缓解。

## 治疗

### 保守治疗
- NSAIDs + 对乙酰氨基酚
- 严重根性症状：短期口服类固醇
- 物理治疗：针对姿势异常，强化腹肌，处理髋屈肌和腰椎旁肌紧张
- **多数患者有慢性腘绳肌紧张**
- 超声和腰椎牵引可缓解反应性肌痉挛
- 严重腿痛：透视引导下硬膜外类固醇注射
- 腰骶支具可短期使用

### 手术适应证
非手术治疗**6周至6–12个月**失败后才考虑。

### ⚠️ 关键循证发现

1. **减压可能恶化结局**（Carragee RCT）：
   - 未减压组临床成功率 96%
   - 减压组仅 66%
   - 减压组假关节率 22%，未减压组 0%

2. **手术复位存在争议**（Poussa 14.8年随访）：
   - 复位内固定组 ODI 7.6 vs 原位融合组 1.6（p<0.01）
   - **原位融合组显著优于复位组**
   - 复位组畸形常见复发

3. **环周融合最优**（Helenius 17年随访）：
   - 环周融合组 VAS背痛仅 5.5mm，无患者报告静息痛或服用镇痛药
   - 后外侧融合 VAS背痛 22.6mm，前路融合 24.1mm

4. 手术 vs 非手术（RCT）：
   - 2年时手术组显著优于非手术组
   - **9年后优势减弱**
